# 🤖 Advanced AI Chatbot

## Description

This project is a rule-based AI chatbot developed using Python. It demonstrates basic Natural Language Processing (NLP) concepts and provides an interactive chatbot experience through both the command line and a simple Flask web interface.

## Features

- 🤖 Responds to greetings
- 🐍 Answers Python-related questions
- 🧠 Explains basic Artificial Intelligence concepts
- 📅 Displays the current date and time
- 😂 Tells programming jokes
- 📋 Shows available commands
- 🗄️ Stores chat logs using SQLite
- 🌐 Runs a simple web interface using Flask

## Technologies Used

- Python
- NLTK
- Flask
- SQLite

## Project Files

- `chatbot.py` – Main chatbot program
- `app.py` – Flask web application
- `database.py` – SQLite database setup
- `chat_logs.db` – Chat history database
- `requirements.txt` – Project dependencies

## How to Run

1. Install the required libraries:
   ```
   pip install -r requirements.txt
   ```

2. Run the chatbot:
   ```
   python chatbot.py
   ```

3. Run the Flask web application:
   ```
   python app.py
   ```
