# 🤖 Advanced AI Chatbot

## Description

This project is a rule-based AI chatbot developed using Python.

It can:

- Respond to greetings
- Answer Python questions
- Explain Artificial Intelligence
- Show current date and time
- Tell programming jokes
- Display available commands
- Store chat logs using SQLite
- Run a simple web interface using Flask

## Technologies Used

- Python
- Flask
- SQLite
- NLTK

## Project Files

- app.py
- chatbot.py
- database.py
- chat_logs.db
- requirements.txt
